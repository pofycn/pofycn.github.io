HEXO Meterial中标签云以及时间轴的相关配置

官方文档地址

官方文档中关于标签云以及时间轴入口的相关配置描述不够详细导致我配置该内容的时候在坑里趴了很久，官网中说是要参考独立页面的配置

然而：



看到这里，我就是这个表情

好吧言归正传，接下来讲解配置标签云以及时间轴：
为了方便（懒）我就把两个东西放到一块直接写了吧

First

cd至blog的相关目录，然后:

hexo new page "tags"
hexo new page "timeline"

执行完以上两条命令后，会在你的blog目录下新建两个文件夹tags和timeline，每个文件夹中都有一个index.md文件。

Second

编辑两个文件夹中的index.md文件

tags中的index.md写入以下内容：

---
title: tags(这个标题可以修改)
date:
layout: tags(这个布局属性不可修改)
---

timeline中的index.md写入以下内容：

---
title: timeline(这个标题可以修改)
date:
layout: timeline(这个布局属性不可修改)
---

Third

这个也是官方文档中没有说清楚的地方，就是添加标签云以及时间轴的入口，这里需要编辑主题的配置文件：
在pages中添加以下内容然后保存：

pages:
        标签云: 
            link: "/tags"
            icon: cloud_circle
            divider: false
        时间轴: 
            link: "/timeline"
            icon: playlist_play
            divider: false

上面配置中：
link：指的是生成页面之后标签云以及时间轴入口的相应链接
icon：为其标题之前的图标，相关图标以及其配置内容可以在https://material.io/icons/中找到。
divider：为是否在两个功能的入口样式中添加分隔的样式

Last

以上就完成了该内容的相关配置，输入下面的命令即可完成并在本地预览效果：

hexo clean
hexo generate
hexo server
